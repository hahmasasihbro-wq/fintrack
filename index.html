Hubungkan ke Google Sheets
Paste URL Web App dari Google Apps Script lu di sini biar data sinkron ke Sheets.
Cara setup: Buka Google Sheets → Extensions → Apps Script → paste isi file Code.gs → Save → Deploy as Web App → copy URL-nya ke sini.
